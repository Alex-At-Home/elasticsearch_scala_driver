case class ElasticsearchDriver(urls: List[String] = List("http://localhost:9200"), connectTimeout: String = "1 second", socketTimeout: String = "10 seconds", retryTimeout: String = "10 seconds", numThreads: Int = 1, basicAuth: Option[(String, String)] = None, defaultHeaders: List[String] = List(), advancedConfig: List[(HttpAsyncClientBuilder) ⇒ HttpAsyncClientBuilder] = List()) extends Product with Serializable

The Elasticsearch driver for the JVM Run start() to provide a client that can be used to execute resource operations

urls

A list of URL endpoints in the format "http://host:port" or "https://host:port"

connectTimeout

The connect timeout (human readable format, eg "1 second")

socketTimeout

The socket timeout (human readable format, eg "10 seconds")

retryTimeout

The retry timeout (human readable format, eg "10 seconds")

numThreads

The number of threads

basicAuth

Basic authentication pair (user, password)

defaultHeaders

A set of default headers applied to every request

advancedConfig

Advanced configuration for the underlying REST client, expressed as a mapping from HttpAsyncClientBuilder to itself, enabled the chained setXxx(...) calls to be performed
